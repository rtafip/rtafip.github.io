Haskell 代码片段

Mar 5, 2017

一些零碎的代码,摘自 《 Haskell 趣学指南》，包含常用函数等。

succ

succ 返回后继

max

max 返回最大值

min

min 返回最小值

!!

!! 按索引获取元素

head

head 取列表的第一个元素

tail

tail 取列表除了第一个元素的所有元素

last

last 取列表的最后一个元素

init

init 取列表除了最后一个元素的所有元素

inits

inits 重复取 init

tails

tails 重复取 tail

length

length 获取列表的长度

null

null 判断列表是否为空

reverse

reverse 反转一个列表

take

take 拿取一定数量的元素

drop

drop 抛弃一定数量的元素

maximum

maximum 返回列表最大值

minimum

minimum 返回列表最小值

sum

sum 列表求和

product

product 列表求积

elem

elem 判断元素是否存在

elemIndex

elemIndex 返回第一个指定元素索引

elemIndices

elemIndices 返回所有指定元素索引

find

find 查找符合条件的元素

findIndex

findIndex 返回符合条件的元素索引

cycle

cycle 循环拼接列表

repeat

repeat 重复指定元素

zip

zip 压缩函数，类似拉链

zipWith

zipWith 用指定的方式 zip

zip3

zip3 是 zip 的 3 个参数版本

zipWith3

zipWith3 是 zipWith 的 3 个参数版本

map

map 对列表每个元素应用函数

filter

filter 过滤出列表中复合条件的元素

takeWhile

takeWhile 一直 take 直到条件不满足

dropWhile

dropWhile 一直 drop 直到条件不满足

intersperse

intersperse 在列表元素之间插入指定元素

intercalate

intercalate 将一个列表插入到另一个列表元素之间

transpose

transpose 转置矩阵

concat

concat 拍扁列表

concatMap

concatMap 先 map 后 concat

and

and 返回列表是否全是 true

or

or 返回列表是否存在 true

any

any 返回列表是否存在元素满足条件

all

all 返回列表是否全部元素满足条件

iterate

iterate 返回重复迭代的结果

splitAt

splitAt 在指定位置分隔数组

span

span 将列表分割为满足条件的列表和不满足条件的列表

break

break 在满足条件时分割列表

partition

partition 类似 span 和 break 不过会对整个列表进行过滤。

sort

sort 列表排序

group

group 列表分组

isPrefixOf

isPrefixOf 判断是否为前缀

isInfixOf

isInfixOf 判断是否为中缀

isSuffixOf

isSuffixOf 判断是否为后缀

lines

lines 将字符串拆为多行

unlines

unlines 与 lines 相反

words

words 将字符串拆成多个词组

unwords

unwords 与 words 相反

nub

nub 移除重复元素，获得最小核心

delete

delete 移除第一个指定元素

\

\\ 去除共同部分

union

union 集合并

intersect

intersect 集合交

fst

fst 获取二元组的第一个元素

snd

snd 获取二元组的第二个元素

flip

flip 返回函数的翻转版本

when

when 相当于没有 else 的 if

sequence

sequence 执行一系列的输入输出操作

mapM

mapM 先进行 map 再遍历返回的 Monad

mapM_

mapM_ 类似 mapM 只是不关心返回的结果
