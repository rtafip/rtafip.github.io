你离异步只有一个语法糖的距离。

syntax sugar

使用 Promise 编写 JavaScript 异步代码已经司空见惯。

但是 Promise 写多了还是觉得有些繁琐。

对此，有了一个新的语法糖提案。

那就是 async 和 await 。

async

async 用于标识一个函数为异步函数。被标识的函数返回一个 Promise 。

也就是 async 其实只是一个语法糖。

内部的实现还是 Promise 。

相当于返回 Promise 的同步函数。

await

await 用于等待一个异步函数。

也就是用于取出 Promise 的值。

因为底层还是用 Promise 实现的。
