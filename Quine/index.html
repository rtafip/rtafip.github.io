大家一看，原来还可以这么玩。于是一发不可收拾，更多语言的版本出现了。

  • Haskell → Python → Ruby
  • Python → Bash → Perl
  • C → Haskell → Python → Perl
  • Haskell → Perl → Python → Ruby → C → Java
  • Ruby → Python → Perl → Lua → OCaml → Haskell → C → Java → Brainfuck → Whitespace → Unlambda

最后有一个聪明绝顶丧心病狂的人做出了100个语言的Quine 。好了好了你赢了。
